silvery Canvas Playground

React components rendered to HTML5 Canvas via the silvery rendering pipeline

How It Works

silvery is a React renderer that computes layout before content rendering. Components know their size during render via useContentRect(), not after.

  • React reconciler builds an InkxNode tree
  • Flexx (pure JS flexbox) computes layout synchronously
  • Content renders to an OffscreenCanvas buffer
  • Buffer is drawn to the visible canvas element
  • requestAnimationFrame drives re-renders on state changes

Canvas Adapter API

import { renderToCanvas, Box, Text, useContentRect } from 'silvery/canvas'; function App() { const { width, height } = useContentRect(); return ( <Box borderStyle="single"> <Text> {width}px x {height}px </Text> </Box> ); } const canvas = document.getElementById('canvas'); renderToCanvas(<App />, canvas);

Render Targets

silvery supports multiple render targets through its RenderAdapter interface:

  • Terminal — ANSI character grid (production)
  • Canvas 2D — pixel buffer (this demo)
  • DOM — accessible HTML elements
  • WebGL — GPU-accelerated (planned)
