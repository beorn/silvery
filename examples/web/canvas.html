inkx Canvas Adapter

React components rendered to HTML5 Canvas via inkx

How it works

  • React components use <Box> and <Text> from inkx
  • useContentRect() returns pixel dimensions during render
  • Layout computed by Flexx (pure JS flexbox)
  • Rendered to OffscreenCanvas, then drawn to visible canvas

Code

import { renderToCanvas, Box, Text, useContentRect } from 'inkx/canvas';

function App() {
  const { width, height } = useContentRect();
  return (
    <Box borderStyle="single">
      <Text>Size: {width}px × {height}px</Text>
    </Box>
  );
}

renderToCanvas(<App />, canvas);
