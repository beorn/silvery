inkx DOM Adapter

React components rendered to DOM elements via inkx - try selecting text!

DOM Advantages

  • Native text selection
  • Screen reader accessible
  • Browser font rendering
  • CSS hover/focus states
  • DevTools inspection

Canvas Advantages

  • Pixel-perfect control
  • Single DOM element
  • Faster batch updates
  • Export to image
  • WebGL upgrade path

Code

import { renderToDOM, Box, Text, useContentRect } from 'inkx/dom';

function App() {
  const { width, height } = useContentRect();
  return (
    <Box borderStyle="single">
      <Text>Size: {width}px × {height}px</Text>
    </Box>
  );
}

renderToDOM(<App />, container);
