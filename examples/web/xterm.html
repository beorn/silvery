silvery xterm.js Adapter

React components rendered to xterm.js via ANSI escape sequences

xterm.js Advantages

  • Full ANSI/VT100 escape sequence support
  • True terminal rendering in the browser
  • Reuses silvery terminal adapter (same as native TUI)
  • Copy/paste, selection, accessibility built-in
  • WebGL renderer available for performance

How It Works

  • silvery terminal adapter produces ANSI diff strings
  • xterm.js parses and renders the ANSI sequences
  • Same output as a real terminal (Kitty, iTerm, etc.)
  • Layout computed by Flexture (pure JS flexbox)
  • Incremental updates via buffer diffing

Code

import { Terminal } from "@xterm/xterm";
import { renderToXterm, Box, Text, useBoxRect } from '@silvery/term/xterm';

function App() {
  const { width, height } = useBoxRect();
  return (
    <Box borderStyle="single">
      <Text>Size: {width} cols x {height} rows</Text>
    </Box>
  );
}

const term = new Terminal({ cols: 80, rows: 24 });
term.open(container);
renderToXterm(<App />, term);
