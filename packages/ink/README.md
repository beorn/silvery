# @silvery/ink

Ink and Chalk API compatibility layer for migrating existing Ink applications to silvery.

> **Note:** This is an internal package. Use `silvery/ink` and `silvery/chalk` imports instead.

```console
$ npm install silvery
```

```tsx
import { render, Box, Text } from "silvery/ink"
import chalk from "@silvery/ink/chalk"
```
