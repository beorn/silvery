# Live Demo

Interactive demos are coming soon. In the meantime, run them locally:

```bash
bunx silvery examples
```

Or clone and run individual examples:

```bash
git clone https://github.com/beorn/silvery
cd silvery
bun install
bun examples/layout/dashboard.tsx
```
